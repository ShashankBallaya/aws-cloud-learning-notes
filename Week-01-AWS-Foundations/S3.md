## What Is It?
S3 is a cloud storage service where you can store and retrieve files from anywhere over the internet

## How It Works (Key Concepts)
- Bucket: A container to store objects (files) in S3.
- Object: A file along with its metadata stored inside a bucket.
- Region: Physical location where your bucker is created.
- Static Website Hosting: Feature to serve HTML/CSS/JS files directly from S3.
- Bucket Policy: JSON-based permission to control access to the bucket and objects.
- Public Access Block: Safety setting to prevent accidental public exposure.
- Versioning: Keeps multiple versions of an object to recover from accidental changes or deletions.
- Storage Classes: Different cost tiers like Standard, IA (Infrequent Access), and Glacier.
- Lifecycle Rules: Automates the transition or deletion of objects based on time.
- CLI Access: Manage S3 using AWS CLI commands instead of the console.
